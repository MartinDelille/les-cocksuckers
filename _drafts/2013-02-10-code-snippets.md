---
title: Code Snippets
---

Whenever you need to post a code snippet, use the liquid tags `hilight` and `endhilight`.

<!--more-->

like this:

{% highlight css %}
nav a:hover {
  color: rgba(0,0,0,.72);
}
nav a.current {
  color: rgba(0, 0, 0, .72)
}
.subtitle {
  margin: 30px 0;
}
{% endhighlight %}
